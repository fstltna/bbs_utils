# BBS Utilities
Command-line utilities for the SBBS BBS software
Official support sites: [Official Github Repo](https://github.com/fstltna/bbs_utils) - [Official Forum](https://synchronetbbs.org/index.php/forum/sysops) 


Included Utilities:
==
- **bbs_announce** (1.1) - This posts a text file to the desired groups. It posts as content, not a attachment! The settings are in the file at the top. Especially note that you need to copy (and edit unless you want to promote my BBS ðŸ˜„) the Announce.txt to /sbbs/exec.
- **perladd.pl** (1.14) - Scans the current folder for new files and submits them to your BBS.
- **updseen.pl** - Updates the perladd.pl /root/.seen file to fix issues - Not usually needed...
- **file_announce** (1.1) - This posts a list of files that you have added using perladd.pl to the message base of your choice. It posts as content, not a attachment! The settings are in the file at the top. Especially note that you need to copy (and edit unless you want to promote my BBS ðŸ˜„) the FilePost.txt & FilePostBottom.txt to /sbbs/exec.
